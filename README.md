## JWT_login
基于 [`JSON Web Token`](http://self-issued.info/docs/draft-ietf-oauth-json-web-token.html) 实现的用户登陆。   
前后端分离，前端采用 `React.js` , 后端采用 `Node.js` + `Koa2`.
